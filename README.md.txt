# PricingLib - Librairie de Pricing d'Options

**PricingLib** est une librairie Python qui permet de pricer des options financières à l'aide de différentes méthodes, notamment la simulation **Monte Carlo**, les options barrières et asiatiques.

## 🚀 Fonctionnalités
- **Simulation Monte Carlo** 📈 pour estimer le prix d'une option.
- **Options Vanilles (CALL, PUT)** 🏦 avec modèles de pricing classiques.
- **Options Barrière** 🚧 (Knock-In, Knock-Out).
- **Options Asiatiques** 📊 basées sur la moyenne du sous-jacent.
- **Tests unitaires** ✅ pour assurer la cohérence des calculs.
- **Support des modèles de payoff modulaires** 🔧.
